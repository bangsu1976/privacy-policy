# 선텍(SunTech) 개인정보처리방침

구글 플레이스토어 앱 개인정보처리방침 링크용 페이지입니다.

## GitHub Pages 배포 방법

1. 이 폴더를 GitHub 저장소로 푸시합니다.
2. 저장소 **Settings** → **Pages** 메뉴로 이동합니다.
3. **Source**에서 **Deploy from a branch** 선택
4. **Branch**를 `main`(또는 `master`), **Folder**를 `/ (root)`로 설정 후 저장
5. 몇 분 후 `https://[사용자명].github.io/[저장소명]/` 주소로 접속 가능합니다.

## 구글 플레이스토어 설정

앱 등록 시 **개인정보처리방침 URL** 필드에 위 GitHub Pages 주소를 입력하세요.

예: `https://username.github.io/privacy-policy/`
